Screen

Command Line
Launch new screen
$ screen [-L]
Use -L to activate logging
Launch new screen with name
$ screen -S <name>
Reconnect to a screen
$ screen -r [#]
If multiple available use the number (or name) to identify which
Reconnect to an attached screen
$ screen -D -r <session id>
List screens
$ screen -ls
Exit a screen
$ exit
Control Commands
Press CTRL+A and the key indicated
Help with control commands
CTRL+A -> ?

CTRL+A -> d

CTRL+A -> x

CTRL+A -> c
